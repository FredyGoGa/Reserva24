# Reserva24 🍷

Reserva24 es una plataforma de comercio electrónico enfocada en la venta de licores a domicilio.

## Características

- Catálogo de productos
- Búsqueda y filtrado
- Carrito de compras
- Checkout online
- Integración con Mercado Pago
- Gestión de pedidos
- Panel administrativo
- Control de inventario

## Tecnologías

### Frontend
- React
- Vite
- Tailwind CSS
- React Router

### Backend
- Node.js
- Express
- Prisma ORM

### Base de Datos
- PostgreSQL

### Servicios
- Mercado Pago
- Cloudinary

## Objetivo

Brindar una experiencia rápida, segura y moderna para la compra de licores online con entrega a domicilio en municipios de la Sabana de Bogotá.
